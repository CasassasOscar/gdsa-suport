This project is maintained by gdsa-upc

The Terrassa Building Challenge 2015 consists in two different tasks to be solved on the Terrassa Buildings 900 dataset:

  • Task 1 Retrieval : Given a query image from a building in Terrassa, search in a collection of images to generate a ranked list of results.
  • Task 2 Classification : Identify which building is represented in the image, if any.
